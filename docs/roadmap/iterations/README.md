# Iterations

Each iteration is a folder under `docs/roadmap/iterations/<iteration-id>/`.

Required files per iteration:

- `iteration.md`
- `execution-plan.md`
- `status-report.md`

Rules:

- `execution-plan.md` is the canonical execution tracker and must use Markdown checkboxes for actionable steps.
- `status-report.md` summarizes progress and validation without duplicating checklist text.
- Closed iterations must not leave open checkboxes in `execution-plan.md`.

Use `docs/roadmap/iterations/ITERATION_TEMPLATE.md` as baseline for new iterations.
